\section*{Project Abstract}

The Project will implement the following features:\\
Front-end:
\begin{enumerate}[label={\arabic*.}]
    \item A login page for user authentication.
    \item A lobby page for the users to enter room number to play chess with friends.
    \item A chess page to play with friends.
    \item A leaderboard page to show how many times do the users win.
\end{enumerate}
Back-end:
\begin{enumerate}[label={\arabic*.}]
    \item Python Flask for server hosting, user authentication, chess mechanics and leaderboard.
    \item Some .txt files simulating a database.
\end{enumerate}
